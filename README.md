隱私權政策

感謝您使用 Latinum（以下簡稱「本應用程式」）。我們非常重視您的隱私權，並致力於保護您的個人資料。

一、資料的蒐集與使用
本應用程式為本地記帳工具，所有使用者輸入的資料（例如收入、支出、分類與備註）僅儲存在使用者的裝置中。
本應用程式不會收集、上傳、傳送或儲存任何個人資料至開發者的伺服器或第三方服務。

二、第三方服務
本應用程式目前未使用任何第三方分析、廣告或追蹤服務。

三、資料安全
由於所有資料皆儲存在使用者裝置本地，資料的安全性取決於使用者裝置本身的安全設定。
本應用程式不會主動存取或傳輸使用者資料。

四、資料刪除
使用者可隨時透過應用程式內的功能刪除所有記帳資料。
刪除後，資料將從裝置中永久移除。

五、兒童隱私
本應用程式並非以兒童為目標對象，亦不會刻意蒐集任何未滿法定年齡之使用者的資料。

六、隱私權政策變更
若本隱私權政策有所變更，將於本頁面更新內容。
